# homebrew-tap

Homebrew tap de apps do [@ananiasfilho](https://github.com/ananiasfilho).

## Frontasks

Lista de tarefas flutuante e sempre-no-topo para macOS.

```bash
brew install --cask ananiasfilho/tap/frontasks
```

Repositório do app: <https://github.com/ananiasfilho/frontasks-app>

> **Nota:** o Frontasks ainda não é notarizado pela Apple. Na primeira abertura o macOS
> pode bloquear — o próprio `brew` mostra o comando para liberar (`xattr -dr
> com.apple.quarantine ...`).
